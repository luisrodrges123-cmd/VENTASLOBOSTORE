# 🐺 VENTAS LOBO STORE - Premium Edition

¡Bienvenido a la versión profesional de **VENTAS LOBO STORE**! Esta es una aplicación Android de alto nivel diseñada con una estética Cyberpunk/Neon y un potente sistema de gestión de ventas integrado con Firebase.

## 💎 Características Premium

*   **📱 Catálogo Profesional**: Galería de productos con efectos visuales modernos y carga rápida de imágenes.
*   **🔐 Panel de Administración Blindado**: Acceso exclusivo mediante contraseña para gestionar inventario y pedidos.
*   **🛒 Pedidos Directos**: Formulario de compra inteligente que recopila datos del cliente y genera un registro automático.
*   **💬 WhatsApp Integration**: Envío de pedidos automáticos a WhatsApp con detalles del producto y vista previa de imagen.
*   **🔔 Notificaciones Push**: Sistema de avisos en tiempo real para nuevos productos usando Firebase Cloud Messaging.
*   **🌐 Ecosistema Web**: Incluye un panel administrativo web sincronizado con la aplicación móvil.

## 🚀 Tecnologías Utilizadas

*   **Lenguaje**: Kotlin (Android) / HTML5 & JS (Web)
*   **Base de Datos**: Firebase Realtime Database
*   **Notificaciones**: Firebase Cloud Messaging (FCM)
*   **UI/UX**: Material 3 Design / Glassmorphism
*   **Imágenes**: Glide Library
